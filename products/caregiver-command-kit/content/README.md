# Content

Draft the Hospital-to-Home Discharge Kit pages here.

Allowed framing:

- Record discharge instructions from the care team.
- Track medication changes as written by the care team or pharmacy.
- Track follow-up appointments, calls, questions, supplies, contacts, and family
  task ownership.

Blocked framing:

- Medical advice, warning signs, diagnosis guidance, treatment steps, dosage
  guidance, recovery promises, "when to call" rules, legal/tax/insurance advice,
  or facility compliance language.
