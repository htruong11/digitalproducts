# Family Caregiver Command Kit

Product workspace for the caregiver MVP described in `docs/imps/IMP-001-init-product.md`.

- `content/` - source copy for the PDF kit.
- `print/` - print/PDF templates and styling.
- `sheets/` - optional Google Sheet structures if the bonus becomes worth building.
- `listing-assets/` - Etsy images and mockups derived from the real product.
- `dist/` - generated outputs; keep this out of git unless a release artifact is intentionally committed.
