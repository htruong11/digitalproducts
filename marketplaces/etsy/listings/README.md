# Listings

Draft Etsy titles, tags, descriptions, FAQs, and post-purchase notes here.

Publication gate:

- Use real screenshots/previews from the actual finished PDF pages.
- Disclose AI assistance in the description when AI helped with copy, layout,
  page structure, mockups, or listing text.
- Include the non-advice disclaimer.
- Keep copy to organizer language: checklist, tracker, log, worksheet, record,
  and template.
- Do not use medical, legal, tax, insurance, probate, HIPAA, compliance,
  treatment, prevention, warning-sign, recovery, diagnosis, dosage, or outcome
  claims.
