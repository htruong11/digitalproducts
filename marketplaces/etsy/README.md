# Etsy

Marketplace workspace for listing copy, competitor research, and support snippets.

Do not store account credentials, payout details, tax information, 2FA recovery codes, cookies, or API tokens here.

- `drafts/` - staged listing packets before human approval or API submission.
- `listings/` - listing copy, tags, FAQs, and post-purchase notes.
- `research/` - competitor snapshots and review mining.
- `support/` - reusable support replies and refund-policy notes.
- `etsyctl/` - thin project-owned API wrapper.
