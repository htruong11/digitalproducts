# IMP-008: Chat Agent Offering

## Decision

Do not build the logged-in/chat-agent product yet. Keep it as an upgrade path
after the PDF library has sales, buyer messages, or repeated requests for
fillable/editable workflows.

## Product Concept

Bundle the research-backed checklists with a site or chat workflow where buyers
can:

- fill out the same worksheet content in a guided flow;
- answer follow-up questions from an assistant;
- export a cleaned-up PDF;
- optionally keep a shared family workspace for updates.

## Purchase Verification Options For Later

- Gumroad license keys: simplest for paid digital files and gated updates.
- Lemon Squeezy: useful if subscriptions or software entitlements become real.
- Shopify digital downloads: useful if the brand moves off Etsy.
- Etsy order verification: possible as a support/manual workflow, but not ideal
  as the primary auth system.
- ChatGPT/Claude handoff: include optional prompt text or file-import
  instructions only after checking platform rules; do not sell raw prompt packs
  as the product.

## Tasks

1. [x] Park the concept under IMP-007 instead of treating it as a current build.
2. [ ] Revisit after 100 qualified views or 10 sales on the PDF library.
3. [ ] If buyers ask for fillable workflows, prototype a no-auth local form or
       Gumroad-gated download before building a full account system.
